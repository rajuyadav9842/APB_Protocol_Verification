# APB_Protocol_Verification

This repository contains the RTL design and UVM-based verification of the AMBA APB (Advanced Peripheral Bus) Protocol using SystemVerilog. The project includes APB master-slave communication, read/write verification, and a complete UVM environment.

## Features

* APB RTL Design
* UVM Testbench
* Read/Write Transactions
* Driver, Monitor, Scoreboard, Agent, Environment
* Functional Coverage
* Waveform Simulation

## Tools

* Verilog/SystemVerilog
* UVM
* QuestaSim
* Cadence Xcelium
